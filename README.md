# D2C Customer Churn Intelligence - Part 1: Data Audit & Exploratory Data Analysis

## Project Objective

The objective of this project is to perform a comprehensive data audit and exploratory data analysis (EDA) on customer, transaction, support, engagement, and intervention data to identify potential churn drivers and generate actionable business insights.

---

## Business Problem

Customer churn directly impacts revenue, profitability, and customer lifetime value (CLV). Understanding the behavioral patterns of customers who churn enables businesses to proactively design retention strategies and improve customer engagement.

---

## Dataset Overview

The project uses six datasets:

* customers.csv
* orders.csv
* support_tickets.csv
* web_events_snapshot.csv
* churn_labels.csv
* intervention_history.csv

---

## Project Workflow
