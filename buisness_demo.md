# Business Memo

## Key Findings

1. Customer inactivity strongly correlates with churn.

2. Support complaints indicate retention risk.

3. High return customers show dissatisfaction.

4. Low website engagement predicts churn.

5. Loyalty participation improves retention.

## Recommendations

1. Target high-recency customers.

2. Improve support response times.

3. Review return-heavy product categories.

4. Create engagement campaigns.

5. Expand loyalty enrollment.
